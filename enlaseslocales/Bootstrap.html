Guía Completa de Bootstrap

¿Qué es Bootstrap?

Bootstrap es un framework CSS y JavaScript desarrollado por Twitter para facilitar la creación de sitios web responsivos y modernos. Ofrece estilos predefinidos y componentes reutilizables como botones, formularios, menús de navegación, carruseles, tarjetas, modales y mucho más.

Con Bootstrap puedes construir páginas que se adaptan automáticamente a distintos tamaños de pantalla (móviles, tablets, computadoras) sin tener que diseñar todo desde cero.

¿Cómo agregar Bootstrap a tu proyecto?

1. Usando CDN (enlaces directos)

Esta es la forma más rápida y sencilla de agregar Bootstrap a tu proyecto. Solo agrega estos enlaces en el <head> y antes de cerrar el </body>:

// En <head> para estilos CSS:
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

// Antes de </body> para scripts JS:
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

2. Usando archivos locales

Descarga los archivos CSS y JS de Bootstrap desde su sitio oficial y guárdalos en tu proyecto. Luego enlaza esos archivos localmente: 

// En <head> (suponiendo que el CSS está en carpeta css):
<link href="css/bootstrap.min.css" rel="stylesheet">

// Antes de </body> (suponiendo que JS está en carpeta js):
<script src="js/bootstrap.bundle.min.js"></script>

Componentes comunes de Bootstrap

Botones

Los botones en Bootstrap vienen con estilos y colores predefinidos que puedes usar fácilmente.

<!-- Botones básicos -->
<button type="button" class="btn btn-primary">Botón Primario</button>
<button type="button" class="btn btn-secondary">Botón Secundario</button>
<button type="button" class="btn btn-success">Botón Éxito</button>
<button type="button" class="btn btn-danger">Botón Peligro</button>

Botón con enlace: Para que un botón funcione como enlace, usa una etiqueta <a> con clases de botón:

Ir a ejemplo.com 
<!-- Botón que es un enlace -->
<a href="https://www.ejemplo.com" class="btn btn-info" target="_blank">Ir a ejemplo.com</a>

Botón de formulario "Enviar": Si quieres un botón para enviar formulario:

<button type="submit" class="btn btn-primary">Enviar</button>

Alertas

Las alertas sirven para mostrar mensajes destacados al usuario, como advertencias, éxitos o errores.

¡Operación exitosa!
Ha ocurrido un error.
Precaución: revise los datos.
Información importante para el usuario.
<!-- Alertas básicas -->
<div class="alert alert-success" role="alert">¡Operación exitosa!</div>
<div class="alert alert-danger" role="alert">Ha ocurrido un error.</div>
<div class="alert alert-warning" role="alert">Precaución: revise los datos.</div>
<div class="alert alert-info" role="alert">Información importante para el usuario.</div>
