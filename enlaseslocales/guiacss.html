Comandos Básicos CSS

1. Cambiar el tamaño de la fuente (letras más grandes)
-- Aumenta el tamaño de la fuente a 30 píxeles
p {
  font-size: 30px; -- Establece el tamaño de la fuente en píxeles
}
2. Centrar texto horizontalmente
-- Centra el texto dentro de un contenedor
.center-text {
  text-align: center; -- Centra el texto horizontalmente dentro del elemento
}
3. Alinear un bloque de contenido horizontalmente (con márgenes automáticos)
-- Alinea un div horizontalmente usando márgenes automáticos
.align-center {
  width: 300px; -- Establece un ancho fijo para el contenedor
  margin-left: auto; -- Aplica un margen automático a la izquierda
  margin-right: auto; -- Aplica un margen automático a la derecha
}
4. Alinear texto verticalmente
-- Alinea el texto verticalmente dentro de un contenedor
.center-vertical {
  display: flex; -- Utiliza Flexbox para alinear contenido
  justify-content: center; -- Centra el contenido horizontalmente
  align-items: center; -- Centra el contenido verticalmente
  height: 200px; -- Establece una altura para ver el efecto
}
5. Cambiar color de texto
-- Cambia el color del texto a rojo
h2 {
  color: red; -- Establece el color del texto como rojo
}
6. Establecer un fondo de color
-- Establece un color de fondo para un contenedor
.container-background {
  background-color: #3498db; -- Establece un fondo de color azul
  padding: 20px; -- Agrega algo de espacio alrededor del contenido
}
7. Cambiar el estilo de la fuente
-- Cambia el estilo de la fuente a cursiva
p.italic-text {
  font-style: italic; -- Aplica un estilo de fuente en cursiva
}
8. Cambiar el espaciado entre letras (tracking)
-- Aumenta el espacio entre las letras
h1 {
  letter-spacing: 5px; -- Aumenta el espaciado entre letras en 5 píxeles
}
9. Colocar una sombra al texto
-- Agrega una sombra sutil al texto
h3 {
  text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.2); -- Aplica una sombra al texto
}
10. Cambiar el color de fondo en función de la posición del cursor
-- Cambia el color de fondo cuando el cursor pasa sobre el elemento
button:hover {
  background-color: #e74c3c; -- Cambia el fondo a rojo al pasar el mouse
}
11. Añadir un borde a un elemento
-- Añade un borde sólido de 2 píxeles a un elemento
div.bordered {
  border: 2px solid #2c3e50; -- Aplica un borde de 2px con color azul oscuro
}
12. Establecer márgenes (espaciado exterior)
-- Establece un margen de 20 píxeles alrededor de un elemento
div.margin-example {
  margin: 20px; -- Aplica un margen de 20px a todos los lados del contenedor
}
13. Establecer relleno (espaciado interior)
-- Establece un relleno de 15 píxeles dentro de un contenedor
div.padding-example {
  padding: 15px; -- Aplica un relleno interno de 15 píxeles
}
