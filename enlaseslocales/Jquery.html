Guía Completa de jQuery

¿Qué es jQuery?

jQuery es una biblioteca (librería) de JavaScript que facilita la manipulación del DOM, manejo de eventos, animaciones y peticiones AJAX con una sintaxis sencilla y compatible con la mayoría de los navegadores.

Permite escribir menos código para lograr efectos y funciones que con JavaScript puro pueden ser más extensos o complicados.

¿Cuál es la diferencia entre jQuery y Bootstrap?

Bootstrap es un framework CSS y JS que ofrece estilos y componentes predefinidos para construir interfaces visuales responsivas.

jQuery es una biblioteca de JavaScript que te ayuda a programar la lógica, manipular elementos HTML, crear animaciones y trabajar con eventos.

¿Se pueden usar juntos? Sí, de hecho muchas versiones antiguas de Bootstrap usan jQuery para sus componentes interactivos (modales, dropdowns, tooltips).

Actualmente Bootstrap 5 no depende de jQuery, pero puedes usar ambas bibliotecas en el mismo proyecto para aprovechar estilos de Bootstrap y la facilidad de manipulación del DOM con jQuery.

¿Cómo agregar jQuery a tu proyecto?

1. Usando CDN

Solo incluye este script antes de cerrar </body> para cargar jQuery desde un servidor público:

<script src="https://code.jquery.com/jquery-3.7.0.min.js"></script>

2. Usando archivo local

Descarga jQuery y guarda el archivo jquery.min.js en tu proyecto. Luego enlázalo así:

<script src="js/jquery.min.js"></script>

Ejemplos básicos de jQuery

Seleccionar elementos y cambiar texto

Este ejemplo cambia el texto de un párrafo al hacer clic en un botón.

Este texto se cambiará.

$(document).ready(function() {
  $("#btnCambiarTexto").click(function() {
    $("#parrafo1").text("¡El texto ha cambiado usando jQuery!");
  });
});

Manejo de eventos

Detectar cuando se pasa el mouse sobre un elemento y cambiar su color.

Pasa el mouse aquí 
$(document).ready(function() {
  $("#caja").hover(
    function() {
      $(this).css("background-color", "#2ecc71");
    },
    function() {
      $(this).css("background-color", "#3498db");
    }
  );
});

Mostrar/Ocultar elementos

Ejemplo de botón que oculta y muestra un párrafo.

Este párrafo puede ocultarse o mostrarse.

$(document).ready(function() {
  $("#btnToggle").click(function() {
    $("#parrafoToggle").toggle();
  });
});

Efectos de animación

Usando fadeIn y fadeOut para animar la aparición y desaparición.

$(document).ready(function() {
  $("#btnFadeIn").click(function() {
    $("#cajaFade").fadeIn();
  });
  $("#btnFadeOut").click(function() {
    $("#cajaFade").fadeOut();
  });
});

Peticiones AJAX

Solicitar datos desde un servidor sin recargar la página.

Aquí se mostrarán los datos cargados. 
$(document).ready(function() {
  $("#btnCargar").click(function() {
    $("#resultado").text("Cargando...");
    $.ajax({
      url: "https://jsonplaceholder.typicode.com/posts/1",
      method: "GET",
      success: function(data) {
        $("#resultado").html("<strong>Título:</strong> " + data.title + "<br><strong>Cuerpo:</strong> " + data.body);
      },
      error: function() {
        $("#resultado").text("Error al cargar los datos.");
      }
    });
  });
});
