Guía Completa de Contenedores con HTML y CSS

1. Contenedor básico con ancho fijo y centrado

Este contenedor tiene un ancho fijo de 600px, está centrado horizontalmente y tiene un fondo gris claro.

<div class="basic-container">
  Contenedor básico con ancho fijo y centrado
</div>
Contenedor básico con ancho fijo y centrado
Cómo modificarlo con CSS:
- Cambiar el ancho: width: 800px;
- Cambiar el color de fondo: background-color: #a0d2eb;
- Cambiar el borde: border: 2px solid #3498db;
- Añadir sombra: box-shadow: 2px 2px 8px rgba(0,0,0,0.2);

🛠 Para modificar este contenedor en CSS: usa .basic-container

2. Contenedor fluido (responsive)

Este contenedor ocupa el 90% del ancho de la pantalla, con un máximo de 900px, para adaptarse a diferentes tamaños de dispositivos.

<div class="fluid-container">
  Contenedor fluido responsive
</div>
Contenedor fluido responsive
Cómo modificarlo con CSS:
- Cambiar el ancho relativo: width: 80%;
- Cambiar el máximo ancho: max-width: 1200px;
- Cambiar color de fondo: background-color: #ffb347;
- Añadir padding extra: padding: 40px;

🛠 Para modificar este contenedor en CSS: usa .fluid-container

3. Contenedor con Flexbox

Usa Flexbox para organizar los elementos hijos en una fila con espacio alrededor y centrados verticalmente.

<div class="flex-container">
  <div>Elemento 1</div>
  <div>Elemento 2</div>
  <div>Elemento 3</div>
</div>
Elemento 1
Elemento 2
Elemento 3
Cómo modificarlo con CSS:
- Cambiar dirección: flex-direction: column;
- Alinear horizontalmente: justify-content: center;
- Alinear verticalmente: align-items: flex-start;
- Espacio entre hijos: gap: 30px;

🛠 Para modificar este contenedor en CSS: usa .flex-container

4. Contenedor con Grid Layout

Organiza los elementos hijos en una cuadrícula de 3 columnas iguales con espacios entre ellos.

<div class="grid-container">
  <div>Celda 1</div>
  <div>Celda 2</div>
  <div>Celda 3</div>
  <div>Celda 4</div>
  <div>Celda 5</div>
  <div>Celda 6</div>
</div>
Celda 1
Celda 2
Celda 3
Celda 4
Celda 5
Celda 6
Cómo modificarlo con CSS:
- Cambiar columnas: grid-template-columns: repeat(4, 1fr);
- Cambiar espacio: gap: 30px;
- Filas automáticas: grid-auto-rows: 100px;
- Alineación: justify-items: center;, align-items: center;

🛠 Para modificar este contenedor en CSS: usa .grid-container
